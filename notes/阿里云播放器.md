# 阿里云播放器

[Web 播放器](https://help.aliyun.com/document_detail/51991.html?spm=a2c4g.11186623.6.942.6cf9480fNX1XNv)定制:

```html
<!DOCTYPE html>
<html>

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, height=device-height, initial-scale=1, maximum-scale=1, minimum-scale=1, user-scalable=no" />
    <title>用户测试用例</title>
    <link rel="stylesheet" href="//g.alicdn.com/de/prismplayer/2.7.4/skins/default/aliplayer-min.css" />
    <script charset="utf-8" type="text/javascript" src="//g.alicdn.com/de/prismplayer/2.7.4/aliplayer-min.js"></script>

    <style media="screen">
        .point {
            position: absolute;
            left: 10px;
            top:  -4px;
            width:  10px;
            height: 10px;
            border: 1px solid blue;
            border-radius: 50%;
            background: red;
            z-index: 100000;
        }
        .point:hover {
            border: border: 1px solid white;
            background: orange;
        }
    </style>
</head>

<body>
    <div class="prism-player" id="J_prismPlayer"></div>
    <script src="http://cdn.bootcss.com/jquery/1.9.1/jquery.min.js"></script>
    <script>
        class PlayTimeComponent {
            timeupdate(event, time) {
                console.log(time.timeStamp);
            }
        }

        var player = new Aliplayer({
            id: 'J_prismPlayer',
            width: '100%',
            autoplay: true,
            // 支持播放地址播放,此播放优先级最高
            source: 'test.mp4',
            // 播放方式二：点播用户推荐
            // vid: '1e067a2831b641db90d570b6480fbc40',
            // playauth: '',
            // cover: 'http://liveroom-img.oss-cn-qingdao.aliyuncs.com/logo.png',
            components: [ PlayTimeComponent ]
        }, function(player) {
            console.log('播放器创建好了。');
        });

        $('.prism-controlbar').append('<div class="point"></div>');

        $(document).on('click', '.point', (event) => {
            event.stopPropagation();
            console.log('click');
        });
    </script>
</body>

</html>
```

服务器端发送请求获取如 playAuth, 视频播放地址等时都需要带上[公共参数](https://help.aliyun.com/document_detail/44432.html?spm=a2c4g.11186623.6.690.16144658dUpmxu).
